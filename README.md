# 🎴 KULO — Kortspelet

Ett multiplayer-kortspel byggt med Vite + Supabase Realtime. Spela med familjen — varje person håller sin egna telefon och ser bara sina kort!

## Kom igång lokalt

```bash
npm install
cp .env.example .env
# Fyll i dina Supabase-uppgifter i .env
npm run dev
```

## Miljövariabler

| Variabel | Värde |
|---|---|
| `VITE_SUPABASE_URL` | Din Supabase projekt-URL |
| `VITE_SUPABASE_ANON_KEY` | Din Supabase anon key |

## Deploy

Se tutorial i README eller deploya direkt med Vercel.
